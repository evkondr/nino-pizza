## NiNo Pizza.

### Description:
NextJS App fpr ordering pizza and other products with paying orders and email notifications.

Create with: TypeScript, Tailwind, Shadcn, Zustand, NextAuth, Prisma, Postgres, YOOKASSA

- RESEND_API_KEY

- YOOKASSA_API_KEY
- YOOKASSA_CALLBACK_URL
